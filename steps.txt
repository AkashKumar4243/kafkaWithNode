1. Download kafka (tar file)
2. extract file
3. copy and paste it in window c
4. rename it kafka
5. open -> kafka -> config -> server (logs.dirs = c:/kafka/kafka-logs)
6. same do for the zookeeper  (dataDir = c:/kafka/zookeeper-data)
7. Start zookeeper 
    a. open cmd at this location 
    b. .\bin\windows\zookeeper-server-start.bat   .\config\zookeeper.properties
8. again open cmd at this location to start kafka server
    a. .\bin\windows\kafka-server-start.bat  .\config\server.properties
9. open cmd at kafka\bin\windows to create topic
    a. kafka-topics.bat --create --bootstrap-server localhost:9092 --topic topic_name
10. To start producer 
    a. kafka-console-producer.bat --broker-list localhost:9092 --topic topic_name
11. To start consumer 
    a. kafka-console-consumer.bat --topic topic_name --bootstrap-server localhost:9092 --from-begining
